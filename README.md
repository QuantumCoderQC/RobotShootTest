# RobotShootTest

This example was made using Armory3D Engine.

Objective: Press `Space` to spawn enemy cube. Shoot the enemy cubes 5 times to destoy.

Controlls:

-Space: Spawn a enemy Cube

-Left Click: Shoot

-W: Move Forward

-S: Move Backward

-A: Move Left

-D: Move Right

-Q: Move UP

-E: Move Down
